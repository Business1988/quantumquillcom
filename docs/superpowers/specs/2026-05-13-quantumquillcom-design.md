# QuantumQuillCom Design Spec

## Goal

Create a mobile software studio website for QuantumQuillCom that feels like a research journal or editorial publication rather than a conventional studio homepage. The studio focuses on mobile software development and IAA advertising monetization.

## Brand and Contact

- Domain: quantumquillcom.com
- VIP: vip@quantumquillcom.com — Premier Client Desk
- Business: business@quantumquillcom.com — Business Partnerships
- Support: support@quantumquillcom.com — Customer Support Team

## Visual Language

- Tone: journal, editorial, considered, research-driven
- Palette: deep wine red, parchment beige, muted copper
- Layout: long-form sections, magazine pacing, serif-led headlines, note columns
- Avoid: dashboard blocks, generic agency hero, uniform card grids

## Structure

- `index.html` — journal-style landing page
- `journal.html` — studio point of view
- `capabilities.html` — service and product capabilities
- `iaa-model.html` — monetization operating model
- `contact.html`
- `privacy.html`
- `support.html`

## Differentiators

- Reads like a studio publication
- More narrative and essay-like
- Strong section dividers and pull-quote behavior
